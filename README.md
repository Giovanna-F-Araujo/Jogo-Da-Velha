# Jogo Da Velha
 Projeto: Criando o jogo da velha com Pyhton 
 
 ### Sobre o jogo:
  - Dois jogadores;
  - Jogam alternadamente preenchendo cada um dos espaços vazios;
  - Elementos: X ou O;
  - Vence o jogador que conseguir formar primeiro uma linha com três símbolos iguais:
    - A linha seja: 
      - Horizontal;
      - Vertical;
      - Diagonal.
